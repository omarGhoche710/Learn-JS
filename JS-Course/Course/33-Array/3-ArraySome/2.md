EX:
عندك قائمة scores بالـ DOM. استخدم .some() تتحقق إذا في 
score فيها textContent أكبر من أو يساوي 50. إذا صحيح، لوّن كل العناصر اللي تحقق الشرط بـ green class passing.

DOM — id="score-list"
23
41
67
15
89
💬 Hint: el.textContent بيرجع String — استخدم Number() أو parseInt() تحوّله 
---
Solution:
```js
let checkScore = [...document.querySelectorAll('.score-item')]
  .some((e) => {
    if (+e.textContent >= 50) {
      e.classList.add('passing'); // side effect — لوّن العنصر
      return true;                // ← هاد اللي يحكيلـ .some() "لقيت"
    }
    return false;
  });

console.log(checkScore); // → true
```

EX:
عندك form inputs بالـ DOM. استخدم .some() تتحقق إذا في input فارغ value.trim() === "". إذا في، ضيف class empty-err على كل input فارغ واطبع تحذير بالـ console.

DOM — class="dom-input"

💬 Hint: لما .some() يلاقي empty input، ما بس ترجع true — ممكن تعمل side effects جوا الـ callback مثل إضافة class

```js
let hasEmpty = [...document.querySelectorAll('.dom-input')].some((e) => {
  if (e.value.trim() === "") {
    e.classList.add("empty-err");
    console.warn("Warn");
    return true;
  }
  return false;
});
```
