أكيد 😄
هاي أمثلة حلوة تبيّن كيف نستخدم `prototype` مع عناصر الـ DOM بدل ما نكرر نفس الكود كل مرة.

الفكرة: نضيف ميثود جديدة لكل عناصر HTML أو لنوع معيّن منها.

---

## ✅ مثال 1: إضافة ميثود تخفي العنصر

بدل ما تكتب `element.style.display = "none"` كل مرة:

```js
Element.prototype.hide = function () {
  this.style.display = "none";
};

// الاستخدام
document.getElementById("box").hide();
```

`this` هون بتمثل العنصر اللي استدعيت عليه الميثود.

---

## ✅ مثال 2: إظهار العنصر

```js
Element.prototype.show = function () {
  this.style.display = "block";
};

document.querySelector(".card").show();
```

---

## ✅ مثال 3: تبديل لون الخلفية

```js
Element.prototype.setBg = function (color) {
  this.style.backgroundColor = color;
};

document.querySelector("p").setBg("yellow");
```

---

## ✅ مثال 4: إضافة class بسهولة

```js
Element.prototype.addClass = function (className) {
  this.classList.add(className);
};

document.querySelector("h1").addClass("active");
```

---

## ✅ مثال 5: حدث click جاهز

تخلي أي عنصر يعمل click handler بطريقة مختصرة:

```js
Element.prototype.onClick = function (callback) {
  this.addEventListener("click", callback);
};

document.getElementById("btn").onClick(() => {
  alert("Clicked!");
});
```

## ⚠️ ملاحظة مهمة (احترافية شوي)

التعديل على `Element.prototype` قوي… بس لازم تستخدمه بحذر لأنه:

* بيأثر على كل عناصر الصفحة
* ممكن يتعارض مع مكتبات تانية

لو بدك أسلوب أنظف، بنعمل class أو function helper بدل prototype.

---
### The Solution

الحل هو اللي قلته — **بدل ما تعدل على prototype مش ملكك، اعمل wrapper**:

**Option 1: Helper Function**
```js
function addHighlight(el, color) {
  el.style.background = color;
}

addHighlight(document.querySelector('div'), 'yellow');
```

**Option 2: Class Wrapper**
```js
class MyElement {
  constructor(selector) {
    this.el = document.querySelector(selector);
  }
  highlight(color) {
    this.el.style.background = color;
    return this; // عشان تقدر تعمل chaining
  }
  hide() {
    this.el.style.display = 'none';
    return this;
  }
}

new MyElement('div').highlight('yellow').hide();
```
