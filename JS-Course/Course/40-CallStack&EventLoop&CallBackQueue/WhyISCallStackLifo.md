## 🧠 ليش الـ Call Stack بيشتغل LIFO ؟

لأن طريقة تنفيذ الدوال (functions) بتفرض هالمنطق.

👉 آخر دالة تُستدعى هي أول دالة تخلص تنفيذها
👉 لذلك لازم تنشال أول شي من الـ stack

---

## 🎬 شوف التنفيذ الحقيقي

```js
function one() {
  two();
}

function two() {
  three();
}

function three() {
  console.log("Hello");
}

one();
```

---

## 🧱 شو بصير داخل Call Stack

### 1️⃣ استدعاء `one()`

```
one
```

### 2️⃣ `one` بتستدعي `two()`

```
two
one
```

### 3️⃣ `two` بتستدعي `three()`

```
three
two
one
```

### 4️⃣ `three` تخلص → تنشال أولاً

```
two
one
```

### 5️⃣ `two` تخلص

```
one
```

### 6️⃣ `one` تخلص

```
فارغ
```

---

## 💡 المنطق ببساطة

الدالة اللي فوق stack هي:
✔ آخر وحدة دخلت
✔ أول وحدة لازم تخلص
✔ لأن باقي الدوال ناطرة نتيجتها

لو ما كان LIFO:
❌ الدوال ما بتعرف ترجع لمين
❌ التنفيذ بيخرب

---

## 🧠 تشبيه سريع

متل كومة كتب 📚
آخر كتاب حطيته فوق… هو أول كتاب بتشيله.
