## ما هو `<dialog>` ؟

عنصر HTML جاهز لعرض **نافذة منبثقة (Popup)** داخل الصفحة.

يُستخدم لعرض:
✅ رسالة مهمة
✅ تأكيد عملية
✅ نموذج إدخال
✅ تنبيه للمستخدم

---

## نوعان من الـ dialog

### 🔒 Modal

يفرض على المستخدم التعامل معه أولًا
لا يمكن التفاعل مع الصفحة خلفه

تفتحه بـ:

```js
dialog.showModal();
```

---

### 🔓 Non-Modal

يظهر فوق الصفحة لكن لا يمنع التفاعل معها

تفتحه بـ:

```js
dialog.show();
```

---

## إغلاق النافذة

```js
dialog.close();
```

---

## الفكرة العملية الكاملة

### فتح

```js
openBtn.addEventListener("click", () => {
  dialog.showModal();
});
```

### إغلاق

```js
closeBtn.addEventListener("click", () => {
  dialog.close();
});
```

---

## متى أستخدمه؟

تستخدم dialog عندما تريد:
👉 لفت انتباه المستخدم
👉 طلب قرار مهم
👉 عرض محتوى مؤقت فوق الصفحة

---

## الخلاصة السريعة

dialog = نافذة منبثقة جاهزة
showModal() = يمنع التفاعل مع الصفحة
show() = لا يمنع التفاعل
close() = إغلاق
