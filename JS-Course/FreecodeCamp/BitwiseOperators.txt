1- AND (&) (
  return 1 If two 1
  ex:
  1 & 1 = 1
  1 & 0 = 0
)

2- OR (|) (
  return 1 if one of them is 1
  ex:
  1 | 0 = 1
)

3- XOR (^) (
  return 1 if two are diffrence
  ex:
  1 ^ 0 = 1
)

4- NOT (~) (
  convert the bit
  ex:
  ~5 = -6
)

5- LEFT SHIFT (<<) (
  101 => 1010
  5 << 1 = 10
)

6- LEFT SHIFT (>>) (
  101 => 010
  5 >> 1 = 2
)
